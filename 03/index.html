<br> #### Week 3: Hand tools and fabrication, Mechanical design Achieved: I created a kinetic sculpture in form of a rotating shadow lamp. I want to give credit to the Thingiverse design by user michael3 (https://www.thingiverse.com/thing:1462353). I used that design to cut the material into geometric shapes. Initially I used this design file to cut cardboard using the laser cutter: ![image of design 1](./design1.png) Then I found the little press-fit joiners flying everywhere in the laser cutter, obstructing the future cutting sequence. So I moved to a different material: wood. Initially, I thought it was success but I quickly found that it did not cut through all the way. I then realized that I did not set the thickness on the software correctly. ![image of design 2](./design2.png) ![image of design 3](./design3.png) I did not want to waste the wood, so I turned it into a base for the motor. ![image of motor](./design4.png) ![image of drill](./design5.png) ![image of motor2](./design7.png) Then, I successfully printed a cardboard utilizing this design. This design had no joiners which meant it would mitigate the joiners flying everywhere problem (sourced from the same link above): ![new design](./design6.png) I started to build the lower half of the shadow lamp from the successful cardboard laser cut materials. I utilized scotch tape to hold the parts together before I used a hot glue gun to provide a little more support: ![image of working product](./design8.png) ![image of working product](./design9.png) ![image of working product](./design10.png) This is when I started on the second laser cut cardboard batch. After letting it run for a few minutes, I checked on it. This is when I saw that it caught on fire: ![fire](./design11.png) ![fire](./design12.png) I was able to salvage one material from the fire. My hypothesis is that since my design required many close laser cuts, the material got too hot and caught on fire. My next cutting iteration, I took close supervision to make sure that I am present to handle the situation if it catches on fire again. ![fire](./design13.png) Then the computer went through a software update and stopped the cutting process mid-way. However, I was able to salvage a few pieces and along with the wood pieces, I was able to complete the project. <video controls> <source src="video.mp4" type="video/mp4"> </video> I left the one face open on purpose so I can easily put in a light bulb in the future. I used Arduino Uno as my microcontroller. The code that I am using is from Nathan's tutorial on the PS70 website on microcontroller. ![fire](./design14.png) A few words to describe this project: Adapt. Overcome. Improvise.
